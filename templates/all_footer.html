a mashup by Steven Lehrburger
with updates by Thiago Ganzarolli (Thanks!)
that currently runs on Google App Engine
and was built using Foursquare's API, Juan Odicio's Foursquare v2 library, bit.ly's API, Google Maps, the gheat-ae/gheat Google Code projects, jQuery, and Blueprint CSS
with generous support and input from Bryan Lence, Jorge Just, John Dimatos, Jorge Ortiz, and Kushal Dave, and MONZY
for the Fall 2009 class Mashups: Remixing the Web at New York University's Interactive Telecommunications Program
and for exhibition at the ITP Winter Show in December 2009
